# DBMS Lab Course

This repository contains my weekly lab assignments and practical coursework for the **Advanced SQL Database Management** course at **IIT Guwahati**. The labs were completed between **Jan. 2026 and April 2026**.

## About the Project
This coursework focuses on practical relational database design, querying, and administration. The files are organized chronologically, containing assignments and scripts from **Week 01 to Week 13**. 

## Key Topics Covered
Throughout the lab sessions, the following concepts were practiced and implemented:
* **Database Design:** Creating schemas, enforcing primary/foreign key constraints, and performing table alterations.
* **Advanced Queries:** Writing complex SQL queries involving advanced joins, nested subqueries, and aggregations.
* **Views:** Implementing views and handling structured DML operations.
* **Database Automation:** Writing stored functions and triggers, including setting up audit log tables.
* **Administration:** Configuring robust MySQL replication.

The files are divided into weekly folders corresponding to the lab sessions:
* `week 01/` through `week 13/`
